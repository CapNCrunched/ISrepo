What's the Deal With SEO?



SEO

As far as website recognition goes, SEO is by far the most interesting thing that I have learned about in the class. The fact that there is a way to optimize your web page so that it works the best when crawled by search engines is truly a testament to how far the internet has come. On top of that there are so many benefits involved with SEO incuding page recognition, search engine trust, popularity, and marketing uses that make SEO a must have for web page developers.SEO is a massive topic that just makes your web page better all around. By using SEO guidlines, the page actually becomes better for both the users and search engines alike. Moz.com, the website we used in class, presented all of the major points of SEO in a way that clearly explained the concept to me, a beginner in SEO. In an attempt to give credit where its due, I want to redirect all of the readers to the links on the left hand side which are primarily to Moz.com's eplanations on SEO concepts. With SEO, everyone has a chance, however small it may be, to get their website circulation to internet users as long as they follow SEO guidelines. SEO promotes equal opportunity to web developers which earned it my favorite topic of the semester.
